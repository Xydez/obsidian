Vad har 0,1 M ättiksyra för syrakonstant?

$$\ce{HAc <=> H+ + Ac-}$$

$$K_a=\frac{\ce{[H+]}\ce{[Ac-]}}{\ce{[HAc]}}$$




|            | HAc     | H+  | Ac- |
| ---------- | ------- | --- | --- |
| Före (M)   | 0,1     | 0   | 0   |
| Förändring | - x     | + x | + x |
| Efter (M)  | 0,1 - x | x   | x   |

Vi mäter pH-värdet.

$\ce{pH}=2,87\iff \ce{[H+]}=10^{-2,87}\iff x=10^{-2,87}$

$K_a=\pu{1,84e-5 M}$
$pK_a=\pu{4,73}$
